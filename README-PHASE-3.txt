PHASE 3 REUSABLE PORTAL

Main improvement:
A new central configuration file has been added:

config.js

From now on, to reuse this website next year, edit only config.js for:
- Event title
- Academic year
- Hosted by
- Categories
- Championship dates
- Registration link
- Apps Script API URL
- Fixture PDF links
- Download PDF links
- Venue Google Map link
- Contact details

Upload/replace these files in GitHub:
- config.js
- index.html
- fixtures.html
- settings-guide.html
- all other existing HTML files may also be uploaded together

How to update next year:
1. Open config.js in GitHub.
2. Click edit.
3. Change title, year, dates, links.
4. Commit changes.

New guide page:
settings-guide.html

This page helps you verify which API URL and registration URL are currently being used.
