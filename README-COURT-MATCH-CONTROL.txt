MATCH TAB UPDATED

Admin > Matches now has four separate sections:
- Live Match Control – Court 1
- Live Match Control – Court 2
- Live Match Control – Court 3
- Live Match Control – Court 4

Each court has:
- Match ID
- Category: U-14, U-17, U-19
- Status: Scheduled, Live, Completed
- Date calendar picker
- Time picker
- Current Set: Set 1 to Set 5
- Name of Team A
- Name of Team B
- Set 1 to Set 5
- Match Won By

Removed from the admin match form:
- Timer Start
- Court field
- Sets Won A
- Sets Won B

The system automatically calculates Sets Won A and Sets Won B from the set scores such as 25-18, 22-25, etc.
