Portal Configuration Guide

1. The only file to edit every year

Open config.js to change the year, title, dates, links, venue, downloads, contact details, and which sections are open to the public (the phase block).

2. First-time backend setup (Code.gs)

  1. Create a new Google Sheet, then Extensions → Apps Script.
  2. Paste in Code.gs from this repo.
  3. Run the setup function once — it creates every tab and a default password.
  4. Run setPassword('yourNewPassword') to set your real admin password.
  5. Deploy → New deployment → Web app → Execute as Me, Who has access: Anyone.
  6. Copy the deployment URL into apiUrl below.

3. The admin dashboard

admin.html is where you enter scores, notices, gallery photos, bracket results, medals and awards during the event. It is not linked from the public nav on purpose — bookmark it directly. It is a convenience gate, not a real login system: your data is still only as safe as the password you set in Code.gs.
